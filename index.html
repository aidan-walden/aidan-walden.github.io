SCP-VR

SCP-VR is a game in ongoing development that I started in September of 2018. It is a modern adaptation of SCP: Containment Breach that has the primary goal of supporting VR input.

SCP-VR is built on top of the Unity game engine, and uses the SteamVR API in order to script interactions between the player and their surroundings.

Twitch Rewards Farmer

Twitch Rewards Farmer is a Node.JS console application that uses Google's puppeteer API in order to simulate a Twitch.tv user session in the background.

Twitch Rewards Farmer was made with the primary objective of taking advantage of Overwatch challenge rewards streams.
